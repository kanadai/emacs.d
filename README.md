## Installation

To install, clone this repo to `~/.emacs.d`, i.e. ensure that the
`init.el` contained in this repo ends up at `~/.emacs.d/init.el`:

```
git clone https://github.com/kanadai/emacs.d.git ~/.emacs.d
cd .emacs.d
cask
```
